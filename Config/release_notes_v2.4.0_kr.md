# BIBIM AI v2.4.0 릴리즈 노트

**릴리즈일**: 2026-04-06

## 🚀 코드 생성 품질 개선 + 사용성 향상

### 코드 생성 중 단계별 진행 상태 표시
- 코드 생성 시 "API 문서 검색 중...", "코드 생성 중...", "API 사용법 검증 중...", "코드 검증 중..." 등 **현재 단계가 실시간으로 표시**됩니다.
- 이전에는 "코드 생성 중..."만 표시되어 어느 단계에서 시간이 걸리는지 알 수 없었습니다.

### 자동 수정 성공률 향상
- 코드 검증 실패 시 자동 수정 전략이 시도 횟수에 따라 점진적으로 강화됩니다.
- 1차 시도에서 해결되지 않으면 더 적극적인 수정을 시도하여 **수정 성공률이 향상**됩니다.
- 비즈니스 로직은 변경하지 않도록 수정 범위가 엄격히 제한됩니다.

### Python 노드 입력 포트 자동 조정
- 생성된 코드가 요구하는 입력 개수에 맞게 Python Script 노드의 **입력 포트가 자동으로 조정**됩니다.
- 노드 생성 후 수동으로 포트를 추가할 필요가 줄어듭니다.

### 로그 시스템 개선
- 로그 파일이 `%APPDATA%/BIBIM/logs/` 폴더로 이동하여 사용자 홈 디렉토리를 깔끔하게 유지합니다.
- 로그 파일이 5MB를 초과하면 자동으로 정리됩니다.

### 내부 안정성 개선
- 코드 생성 파이프라인의 내부 구조를 개선하여 안정성과 유지보수성을 높였습니다.
- 불필요한 레거시 코드를 정리했습니다.
- 다국어 지원 범위를 확대했습니다.
