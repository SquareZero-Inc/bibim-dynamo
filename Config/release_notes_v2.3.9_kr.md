# BIBIM AI v2.3.9 릴리즈 노트

**릴리즈일**: 2026-03-19

## 🛡️ AI 코드 품질 대폭 강화

### 더 정확한 코드 생성
- AI가 Revit에서 **제거된 API**(Element.LevelId, ElementId.IntegerValue 등)를 사용하는 코드를 생성하면 **즉시 차단**합니다. 이전에는 경고만 표시되어 런타임 에러가 발생할 수 있었습니다.
- AI가 참고하는 API 문서의 신뢰도 체계를 개선하여, 잘못된 참고 자료로 인한 **할루시네이션(환각) 코드 생성이 감소**합니다.
- 코드 자동 수정 시에도 제거된 API 정보가 반영되어, 수정 과정에서 같은 실수를 반복하지 않습니다.

### 영어 모드 개선
- 영어 빌드에서 명세서 기반 코드 생성 시 **API 문서 검색과 분석 컨텍스트가 정상 적용**됩니다. 이전에는 영어 모드에서 이 기능이 누락되어 코드 품질이 낮았습니다.
- 검증 차단 메시지가 **빌드 언어에 맞게** 표시됩니다 (한국어 빌드 → 한국어, 영어 빌드 → 영어).

### 검증 범위 확대
- **Dynamo 전용 API**(RevitServices, ProtoGeometry, DSCoreNodes)도 검증 대상에 포함되어, 더 넓은 범위의 API 오류를 사전에 잡아냅니다.
- 복잡한 메서드 호출(중첩 괄호)의 인자 수를 정확히 파싱하여 **오탐(false positive) 차단이 감소**합니다.

### 안정성 개선
- API 문서 검색이 일시적으로 실패해도 **자동 재시도** 후 진행합니다.
- API 문서 검색에 **전용 타임아웃(45초)**이 적용되어, 검색이 느릴 때 전체 응답이 지연되지 않습니다.
